# Instalando e rodando Front/Back
### Entra no diretório server
```
cd server
```
### Instala dependências
```
npm i
```
### Gera o prisma
```
npx prisma generate
```
### Compilar
```
ctr+shift+b
```
### Inicia o servidor
```
npm run start
```

### Entra no diretório front
```
cd front
```

### Instala dependências do front
```
npm i
```
### Inicia o servidor do front

```
npm run serve
```
